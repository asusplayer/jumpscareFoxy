# 1/1000 chance for Withered Foxy Jumpscare every second #


! This is a remake/port of the already existing mod jumpscareFoxy by olivr !

## Installation ##

put the jumpscareFoxy folder into BepinEx/plugins
Easiest way is to install using a modmanager like r2modman
the mod is very configurable and is fully compatible with REPOConfig.

you can also easily change the jumpscare video and audio by replacing its files for those you want on the assets folder, just make sure to keep the file names and extensions the same as the original.

due to Unity limitations, the jumpscare is made of video frames, so be careful when changing the jumpscare video.
